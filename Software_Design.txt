Data Base Metadata:
  videogames:
    id | title | description | developer | release year | clasification | image

    *PostgreSQL date format: 1922-02-02

API Endpoints:
  /api/videogames
  /api/videogames/<videogame name>
  /api/videogames/dev/<developer name>

Methods:
  GET: to get the information of all the videogames from the Data Base or from only the one specified
  POST: to append a new videogame to de Data Base
  DELETE: to remove the specified videogame from  the Data Base
